# Smart EV Charging Scheduler

A vanilla HTML, CSS, and JavaScript frontend app that simulates an EV charging station queue using CPU scheduling algorithms.

## Features

- Vehicle input form with arrival time, charging time, and battery percentage
- Battery color coding for low, medium, and high charge levels
- FCFS, SJF, Priority, Round Robin, Multilevel Queue, and MLFQ scheduling
- Adjustable Round Robin time quantum
- Execution order, waiting time, turnaround time, and average waiting time
- Visual Gantt chart
- Best algorithm suggestion based on average waiting time
- Sample data and reset controls
